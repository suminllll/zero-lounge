# Zero Lounge (제로라운지)

모임 예약 흐름을 중심으로 UI 구조를 설계·구현하는 개인 프로젝트  
현재는 핵심 사용자 흐름(정보 확인 → 선택 → 신청) 기준으로 화면 구조를 정리하고 단계별로 구현 중

## 목표
- 예약 흐름을 한눈에 이해할 수 있는 화면 구조 설계
- 사용자 행동 동선을 줄이는 UI 구성(정보 우선순위/CTA 배치)
- 재사용 가능한 컴포넌트 구조로 확장 가능한 코드베이스 구성

## 진행 상태
- ✅ 화면 구조/레이아웃 설계
- ✅ 핵심 페이지 라우팅/기본 UI 구성
- 🔄 예약 흐름 UI 구현 진행 중
- ⏳ 배포(예정)


## Tech Stack
- Next.js
- TypeScript
- TailwindCSS
- Vercel
